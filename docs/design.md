## 树初始化
1. 获取初始化数据
2. 预处理数据
    1. 根据 setting 转换节点数据
    2. 转换成树形结构
3. 全量构建节点对象树(树根是节点的子类)
4. 构建 DOM 视图
    1. 获取插入根节点
    2. 后序遍历构造节点 html(或者 DOM 节点)
        1. 根据数据节点展开状态，按需构造节点 DOM
    3. 将生成对象插入 DOM 中
5. 绑定事件处理器，初始化事件代理

## 树添加数据
1. 获取要添加的数据
2. 预处理数据
    1. 转换节点数据
    2. 将数据转换成树形结构
3. 全量构建子节点对象树/森林
4. 按需构造子级 DOM
    1. 获取插入根节点
    2. 后序遍历构造节点 html(或者 DOM 节点)
        1. 根据数据节点展开状态，按需构造节点 DOM
    3. 将生成对象插入 DOM 中
5. 更新父节点状态（递归）。
